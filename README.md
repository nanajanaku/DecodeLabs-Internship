## 🔑 Key Findings
- Total Revenue: **$1.26 million** across 1,200 orders
- Top Products: **Chair & Printer** at ~$195K each
- **41% of orders** cancelled or returned
- **Instagram** top referral source (259 orders)
- **8 outlier orders** detected above $3,330
- **74%** of customers used coupon codes
