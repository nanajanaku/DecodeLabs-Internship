# 🧹 Week 1: Data Cleaning & Preparation
### Decode Lab Data Analytics Internship

---

## 📌 Project Overview
The first step of any analytics project is ensuring 
the data is clean, accurate and ready for analysis. 
This week I cleaned a raw e-commerce dataset by 
handling missing values, duplicates and incorrect 
data formats.

---

## 🎯 Goal
Clean a raw dataset by handling missing values, 
duplicates and incorrect data.

---

## ✅ Key Requirements Completed
- [x] Identified missing or null values
- [x] Removed duplicate records
- [x] Corrected data formats (dates, numbers, text)
- [x] Prepared cleaned dataset ready for analysis

---

## 🛠️ Tools Used
- Microsoft Excel

---

## 📋 Cleaning Steps Performed

### 1. Missing Values
- Identified all null and empty cells
- Handled missing values appropriately
- Documented all changes made

### 2. Duplicate Records
- Checked for duplicate OrderIDs
- Removed all duplicate rows
- Verified final row count

### 3. Data Format Corrections
- Standardized Date column format
- Corrected numeric columns (Revenue, UnitPrice)
- Standardized text columns (Product, OrderStatus)

## 💡 Key Learnings
Data cleaning is the most critical step in any 
analytics project. Dirty data leads to wrong insights 
and wrong insights lead to bad business decisions. 
Clean data = reliable analysis.
